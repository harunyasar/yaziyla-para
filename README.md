# Yaziyla-Para

Para bilgisini yazıya çeviren küçük bir sınıf

## Install

Via Composer

``` bash
$ composer require harunyasar/yaziyla-para
```

## Usage

``` php
$para = new Para(21315.25);
echo $para->text; # yirmi bir bin üç yüz on beş TL, yirmi beş kr.
```
